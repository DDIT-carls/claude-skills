# 女娲  Skill造人术

## 核心理念
女娲不复制人，它提取认知操作系统。
- 这个人用什么心智模型看世界？
- 用什么决策启发式做判断？
- 如何表达自己？
- 绝对不做什么？
- 这个Skill做不到什么？

## 执行流程

### Phase 0: 确认需求
收到名字后确认：对象是谁、聚焦方向、用途目的、是否已有Skill。

### Phase 1: 六路并行调研
同时启动6个Agent：
1. 著作/文章/系统性思考
2. 播客/访谈/即兴思维
3. 社交媒体/表达DNA
4. 外部评价/批评者视角
5. 重大决策记录
6. 人生时间线

### Phase 2: 框架提炼
- 心智模型（3-7个）：跨域重复出现、有预测力、有排他性
- 决策启发式（5-10条）：可表达为"如果X则Y"
- 表达DNA分析
- 价值观与反模式
- 诚实边界

### Phase 3: 构建Skill
输出SKILL.md，包含所有提炼结果。

### Phase 4: 质量验证
- 已知测试：用此人公开回答过的问题测试
- 边界测试：用未讨论过的问题测试，应表现出适度不确定
- 声音测试：100字分析，能否识别出是谁

## 品味准则

好Skill vs 坏Skill：
- 能处理新问题 vs 只能重复旧语录
- 有内部矛盾 vs 观点高度一致（假）
- 承认不确定性 vs 什么都有答案
- 3-7个深刻模型 vs 20条模糊原则

永远不做：
- 捏造此人未说过的话
- 把通用智慧包装成独特洞见
- 忽视负面评价和争议
- 暗示Skill等同于本人
